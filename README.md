## Install

Clone the repo using

```
$ git clone https://github.com/HarrisonHenri/serverless-todo-api.git
```

```
$ cd serverless-todo-api
```

Install all the required dependencies

```
$ yarn
```

Install the dynamodb locally

```
$ serverless dynamodb install
```

Then execute

```
$ yarn dev
```

and

```
$ yarn dynamo:start
```
